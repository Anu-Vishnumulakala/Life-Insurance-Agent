# Life-Insurance-Agent
An AI-powered Streamlit app that calculates life insurance coverage and recommends suitable term-life options using real financial logic and live web research.
## Project Overview
Choosing the right life insurance coverage is confusing for many individuals. Most online calculators act like black boxes and fail to explain *why* a specific coverage amount is recommended.
This project solves that problem by combining **financial modeling** with **AI agents** to deliver:
- Clear, explainable coverage calculations  
- Personalized recommendations  
- Real-world, up-to-date insurance product insights  
## Key Features
- **Coverage Calculation Engine**  
  Calculates recommended life insurance coverage using income replacement and discounted cash flow logic.
- **AI Agent Reasoning**  
  An AI agent explains assumptions, methodology, and results in a structured way.
